# Match Battle (рабочее название)

Match-3 battle игра для круга друзей - духовный наследник Beauties Battle (PalmFun, 2016-2019) в визуальном стиле Ode to Heroes. Без монетизации, для 5-7 игроков на Android.

## Суть

- Доска 7x7, drag-and-connect цепочки из фишек меч/сердце/щит
- Команды 3v3, фракции Огонь/Дерево/Вода + элитные Инь/Ян
- Гача с pity, шарды, экипировка, idle-прогрессия
- PvP: общая доска, ходы по очереди с таймером, цепочка соперника видна вживую (модель Match Masters)

## Стек

- Клиент: Phaser 3 + TypeScript + Vite, вертикальная PWA (фолбэк - APK через Capacitor)
- Игровая логика: чистый TS-модуль без Phaser - работает и в headless-симуляторе баланса, и на сервере
- Сервер (позже): Colyseus + SQLite, аккаунты логин+пароль
- Интерфейс на русском

## Структура

- `docs/research/` - ресерч референсов, механик жанра и стека (июль 2026) + память утерянной первой версии (Anime Match Battle, март 2026)
- `docs/DESIGN.md` - игровой дизайн: все принятые решения

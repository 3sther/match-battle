# core - игровая логика

Чистый TypeScript без зависимостей от Phaser и DOM. Здесь живут доска, цепочки, бой 3v3, фракции, ульты.

Правило: из этого каталога НЕЛЬЗЯ импортировать `phaser` и обращаться к `window`/`document`. Модуль исполняется в трёх средах: клиент (Phaser-обвязка), headless-симулятор баланса (M1), сервер Colyseus (M6-M7).

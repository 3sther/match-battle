# Игровой дизайн - принятые решения

Летопись решений раундов планирования (июль 2026). Обновлять при каждом дизайн-решении - это источник истины при спорах «а как мы договаривались».

## Бой

- **Доска:** 7x7, drag-and-connect (соединение пальцем, включая диагонали), фишки трёх типов: меч (атака), сердце (лечение), щит (защита). Автокаскады душим - победу должен решать игрок, не лавина рандома (инсайт Match Masters).
- **Команды 3v3.** У каждого героя свои HP, atk, def, ульта. Бой до полного уничтожения команды.
- **Меч-цепочка:** урон = сумма atk живых героев × множитель цепочки, через смягчение защитой цели (atk/def-соотношение с капом). Урон идёт в выбранную цель - игрок переключает фокус тапом по портрету врага.
- **Сердце-цепочка:** лечит всю команду, % от max HP каждого, масштабируется от недостающего HP - раненым больше, здоровым меньше.
- **Щит-цепочка:** общий командный щит, скалируется от def команды.
- **Ульты индивидуальные.** Гибридный заряд: шкала (цепочки) + ability tile (появляется за цепочку от N фишек, матч по ней - мгновенный буст заряда всем). Overcharge сохраняем.
- **Заряд по ролям:** цепочка заряжает ульту «своей» роли на 100% (меч → ДД, сердце → саппорт, щит → танк), остальные роли на 40%. Числа - кандидаты на тюнинг симулятором. Так саппорт не остаётся без заряда на «мечевой» доске, а моно-составы (3 ДД) остаются архетипом с встроенной слабостью, не доминантой.
- **TTK:** цель 8-12 ходов на дуэль. Валидируется симулятором, не ощущениями.

## Фракции

- Стандартные: **Огонь → Дерево → Вода → Огонь** (+20% урона по контрируемой). Содержат R/SR/SSR.
- Элитные: **Инь ↔ Ян** (контрят друг друга, нейтральны к тройке - паттерн Celestial/Hypogean из AFK Arena). Только SSR/UR.
- Синергии состава: бонусы за моно-фракцию / радугу, честно показываются до боя.
- Иконки фракций - иероглифы (как в OTH).

## Ростер

- Новый, китайская мифология/история без привязки к эпохе (Нюйва, Сунь Укун, Хуа Мулань, Гуань Юй, Чанъэ...). Старые 19 японских персонажей НЕ переносим.
- Старт: ~16-18 героев. Стандартные фракции по 4 (танк/дд/саппорт/гибрид), элитные по 2-3.
- Ниши вместо лестницы редкости: у R-героя своя роль (пробивание щитов, разгон шкалы...), а не «UR, но хуже». Пассивка + тип ульты + разброс статов.

## Баланс - уроки первой версии

Главная болезнь старого баланса: хил не скалировался (16 × mult, константа), атака скалировалась (15 × mult × atk/10) → на низких atk бои вязкие и лёгкие, на высоких - burst-гонки «кто быстрее убил». Обе жалобы игрока - из одной формулы.

Принципы новой версии:
- все три действия скалируются согласованно;
- баланс тюнится headless-симулятором (10к+ боёв AI vs AI, распределение TTK, винрейты составов) - игроки-миньмаксеры, «на глаз» не пройдёт;
- архетип с винрейтом >55% против сбалансированных составов - крутим до камень-ножницы-бумага.

## PvP (M7)

- Общая доска, ходы по очереди, таймер ~20 сек на ход.
- Цепочка соперника транслируется вживую (путь пальца через websocket с троттлингом).
- Сервер авторитарен: логика боя = тот же TS-модуль, что в клиенте и симуляторе.
- Лидерборд с сезонными сбросами (иначе ранний лидер навсегда впереди и компания разбегается).

## Мета (M4)

- Гача: прозрачный soft/hard pity, шарды (60 = герой), дубликаты = усиление. 
- Экипировка + руны, 3 билда на героя (из первой версии - работало).
- Idle-доход для PvE-фарма - не требует синхронного онлайна всех друзей.
- Клан/гильдия одна на всех - вклад через активность.
- ВЫКИНУТО: VIP, battle pass, многослойная экономика с daily-гейтами - монетизационный мусор.

## Арт (M3)

- Стиль: Ode to Heroes - painterly полу-реализм («рисованный, близкий к реализму, не аниме, не придуманные существа»), ханьфу, приглушённая база + один акцентный цвет, мягкий свет. UI: тёмно-синий + золото, облачный орнамент, ненагруженный.
- Два слоя арта: портрет (гача/карточки) + боевой аватар.
- Генерация локально (RTX 5070 8GB): SDXL через Fooocus, при нехватке качества - Flux GGUF через ComfyUI. Корп. ChatGPT - только черновики композиций.
- Консистентность: единый стиль-блок промпта + IPAdapter с эталоном + один чекпоинт на всю партию.
- Процесс: 1 герой в 3-4 стилях → выбор эталона игроком → лок стиля → массовая генерация. Массовое производство ДО лока стиля запрещено (причина заброса первой версии).

## Технические решения

- Phaser 3 + TypeScript + Vite; игровая логика - отдельный пакет без зависимостей от Phaser.
- PWA вертикальная, `display: fullscreen`; M0 - проверка полноэкранности на реальном телефоне (гейт). Фолбэк - Capacitor APK.
- Сервер: Colyseus + SQLite, аккаунты логин+пароль, без OAuth.
- Хостинг: клиент - Cloudflare/GitHub Pages; сервер - Oracle Cloud Free (бэкап SQLite наружу, деплой одной командой - переезд при бане за 30 минут). Игроки фактически все вне РФ.
- Git: личный GitHub 3sther/match-battle (приватный). Рабочий GitLab не используем - личный проект.

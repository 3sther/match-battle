# Гача и инвентарь — дизайн-документ

## Градации персонажей
- R (обычный), SR (редкий), SSR (супер-редкий), UR (ультра-редкий)
- Каждый из 6 текущих персонажей получает градацию

## Шарды
- 60 шардов = 1 копия персонажа
- В гаче выпадают: 10, 20, 40 шардов (чаще мелкие), реже — полный персонаж
- Дубликат = усиление (повышение статов или звёздности)

## Гача — пул дропа
1. **Шарды персонажей** — 10/20/40 шт, реже полная копия
2. **Экипировка** — оружие, броня, аксессуары разных редкостей
3. **Руны** — 4 слота на персонажа, дают бонусы/бафы
4. **Расходники** — сундуки с бонусами для доски, малое кол-во валюты призыва, шарды обычных персонажей

## Валюта
- Единая валюта (кристаллы/монеты)
- Заработок: за каждые 3 подряд выигранных уровня выдаётся награда
- Стоимость: 1 pull = 180, 10-pull = 1600 (скидка ~11%)

## Инвентарь
- Слоты экипировки на персонажа: оружие, броня, аксессуар + 4 руны (итого 7 слотов)
- Экипировка даёт бонусы к ATK/DEF/HP и спец. эффекты

## Билды
- У каждого персонажа 3 слота билдов
- Билд = набор экипировки + руны
- Если все 3 заняты — нужно перезаписать один из них
- Билды сохраняются между забегами

## Панель администратора
- Доступна из меню "Настройки"
- Редактирование: валюта, шарды, экипировка, уровень персонажей
- Для тестирования и отладки

## UI / Навигация
- Гача — отдельная вкладка в нижнем меню
- Инвентарь — вкладка в нижнем меню
- Настройки — вкладка с админ-панелью
